Tents possiede le seguenti feature:
	t: posiziona una tenda sulla cella su cui è appoggiato il cursore
	g: riempe tutte le caselle bianche con erba
	n: cambio dello schema in avanti(nella lista di schemi)
	b: cambio dello schema indietro(nella lista di schemi)
	r: reset delle mosse fatte
	a: suggerimento per la risoluzione del rompicapo

PS: la feature di autocompletamento ha una difficoltà computazionale troppo elevata per processare griglie che superano
    l'8x8 anche con appropriate funzioni di esclusione delle mosse a priori sbagliate
